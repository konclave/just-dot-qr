This is the library that generates the QR code based on the string input. The QR code consists of circles – that's the main difference comparing to the traditional QR codes. In the middle of the QR code there's a space for the logo.

The string to encode, QR code background color (could be transparent), logo URL/path are provided as an input.
It's a standalone library, but it has a React component wrapper provided to use it with React. 

Use qr-code.svg and qr-code.png as a visual reference
